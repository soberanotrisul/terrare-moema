# Terrare · Moema

Landing page de captação de leads para Meta Ads.

## Como usar

1. Faça o upload do arquivo `index.html` neste repositório
2. Ative o GitHub Pages em **Settings → Pages → Source: main**
3. Acesse o link gerado: `https://seuusuario.github.io/terrare`

## Integração

Os leads preenchidos são enviados automaticamente para o Google Sheets vinculado. A coluna **Projeto** identifica os leads como **TERRARE MOEMA**.
